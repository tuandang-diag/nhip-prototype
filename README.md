# Nhịp Prototype

Runnable Vietnamese prototype for the organizer and member flows defined in
`ideas/08_nhip_group_ops/07_prototype.md`.

## Commands

```bash
npm install
npm run dev
npm run type-check
npm test
npm run build
```

The demo stores state in browser `localStorage`. Use **Đặt lại bản demo** in
the organizer navigation to restore the seeded class and 60 members.
